Capstone Project Case Study

Project Overview

This capstone project is an interactive Three.js-based memorial webpage designed to create emotional connections through 3D elements, animation, and storytelling. The primary objective was to visualize loss, memory, and personalized digital rituals using custom markers and engaging transitions.

Problem

Traditional methods of remembering pets or loved ones often lack personalization and interactivity. The challenge was to use web-based 3D experiences to represent memory in a way that felt emotionally resonant and meaningful for users.

Solution

The final solution involved a Three.js scene with orbiting elements, interactive markers, animated text using GSAP, and a clean visual style. Users could explore memories by scrolling and clicking, triggering animations and model transitions that mimicked emotional depth and personalization.

Process

  • Research & Inspiration: Looked into existing Three.js memorials, digital graveyards, and commemorative apps.
  • Low-fidelity Prototypes: Sketches and simple 3D donut models to test interaction flow.
  • Mid-fidelity: 3D printed elements, laser-cut tests, and GSAP text transitions.
  • User Feedback: Testers were confused in scene 2, distracted in scene 3. Improvements were logged and prioritized.

Results

The project was well received for its emotional tone and smooth transitions. Users were particularly drawn to the typewriter animation and overall visual atmosphere. However, several areas for improvement were identified, including adding clearer UI feedback, improving scene flow, and redesigning narrative structure.

Future Improvements

  • Add highlight and hover feedback to interactive choices.
  • Redesign narrative log to improve coherence.
  • Add animated visualization of user impact at the end.
  • Consider background sounds and voice-overs for context.

Conclusion

This capstone demonstrates how emotion-driven web interactions can provide a space for memory and reflection. By combining 3D environments, storytelling, and animation, the project creates a space that’s personal and engaging—offering new digital rituals for remembrance.
